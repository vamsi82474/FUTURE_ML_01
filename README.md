# FUTURE_ML_01
A Machine Learning project to forecast future sales using historical data, featuring data preprocessing, time-based feature engineering, model evaluation (R², MAE), and visualization of predicted trends.

## Steps
- Data preprocessing
- Feature engineering (time_index)
- Train-test split
- Linear Regression model
- Evaluation (R2, MAE)
- Forecasting

## Business Insight
Sales show steady growth. Model predicts continued increase. Useful for inventory planning.
